Grade 9 Lessons



Topics per quarter:

Hypertext Markup Language

HyperText Markup Language or HTML for short is a standard markup language for web page creation. It is responsible for creating sections, paragraphs, and links with its own elements like tags and attributes. There are quite a few uses for HTML like its use for displaying hyperlinks, media files, web page elements and many more. It can be used for organizing documents and inserting links as well.

Cascading Style Sheets

CSS stands for Cascading Style Sheets. And is the "language" many use to create visually interesting web pages. It describes how HTML elements are to be displayed.

Bootstrap

Bootstrap is a development framework used to create websites and apps. Mainly relied on for mobile websites. It's incredibly easy to use once you learn it and it can help create websites faster thanks to it having the basics for responsive web development.
